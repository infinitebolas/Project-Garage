# Project-Garage
Projet de gestion de garage et de voitures

Etapes de lancement:
1) Cloner le repository
2) Ouvrir la db avec mysql

Lancement des serveurs:
1) Dans le dossier backend, exécuter mvnw.cmd spring-boot:run
2) Dans le dossier frontend/front exécuter ng serve

Ouvrir dans le navigateur le chemin indiqué par angular
